# Driftex

A lightweight PHP rate limiting library for controlling API traffic with precision and simplicity.

---

## 🚀 Features

- Simple and fast rate limiting
- Storage-agnostic architecture
- File-based storage included
- Easy to extend (Redis, DB, etc.)
- PHPUnit tested
